# Data Job Market Analysis Using SQL (2023)

## Overview
This project analyzes global data job postings using SQL to understand salary trends, skill demand, and role-based job market patterns.  
The analysis supports dynamic filtering by job role and location, with demonstrations focused on Data Analyst roles in India.

## Business Questions Answered
- What are the top-paying roles for Data Analysts?
- Which skills are required for high-paying roles?
- What are the most in-demand skills in the job market?
- Which skills are associated with higher salaries?
- What are the most optimal skills to learn based on demand and pay?

## Tools & Environment
- PostgreSQL
- VS Code
- SQL (Joins, Aggregations, Filtering, Grouping)
