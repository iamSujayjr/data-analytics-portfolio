## Project Overview  
This project analyzes the data job market to identify trends related to salaries, job demand, and skill requirements. The interactive dashboard allows users to explore insights by selecting different job roles and locations.

The main goal of this project is to transform raw job posting data into meaningful business insights using professional data modeling and visualization techniques.

---

## Objectives  

- Analyze salary trends across different data-related job roles  
- Identify the most in-demand technical skills in the data job market  
- Compare top-paying data roles using interactive charts  
- Enable role-based and location-based exploration  
- Improve dashboard usability and clarity  

---

## Tools & Technologies Used  

- Power BI Desktop  
- Power Query (Data Cleaning and Transformation)  
- Data Modeling (Table Relationships)  
- DAX Measures  
- Field Parameters  
- Interactive Dashboard Design  

---

## Key Features  

- Dynamic filtering by job role and country  
- KPI summary cards for job count, salary metrics, and skill density  
- Parameter-based chart switching for salary comparison  
- Centralized measures table for better model organization  
- Clean and recruiter-friendly dashboard layout
