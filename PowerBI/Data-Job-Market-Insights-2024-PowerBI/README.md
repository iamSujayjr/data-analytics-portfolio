## Project Overview  
This project analyzes the data job market to identify trends related to salaries, job demand, and skill requirements. The interactive dashboard enables users to explore insights dynamically by selecting different job roles and locations.

The project focuses on transforming raw job posting data into meaningful business insights using professional data modeling, DAX calculations, and interactive visual analytics.

---

## Project Objective  

The main goals of this project are to:

- Analyze salary trends across multiple data-related job roles  
- Identify the most in-demand technical skills in the data job market  
- Compare top-paying data roles using dynamic visualizations  
- Enable role-based and location-based exploration  
- Improve dashboard usability and analytical clarity  

---

## Key Features  

- Interactive slicers for job role and country selection  
- KPI cards displaying job count, salary metrics, and skill density  
- Parameter-based chart switching for dynamic salary comparison  
- Centralized measures table for organized DAX calculations  
- Optimized data model with table relationships  
- Clean and recruiter-friendly dashboard layout  

---

## Tools & Skills Used  

- Power BI Desktop

- DAX Measures

## How to Use  

1. Select a Job Role using the slicer  
2. Choose a Country for location-based insights  
3. Review KPI cards for salary and job volume metrics  
4. Use parameter toggles to switch between salary views  
5. Analyze skill demand and role-based salary comparisons  

---

## Key Insights  

- Salary levels vary significantly across data roles and regions  
- Certain technical and enterprise tools are strongly associated with higher-paying jobs  
- Skill demand does not always correlate directly with salary value  
- Location and role selection significantly impact market insights  

---

## Notes  

- The dashboard uses a structured data model with defined table relationships  
- All KPI calculations are powered by DAX measures  
- Parameters are used to reduce visual clutter and improve analytical flexibility  
- Designed to simulate real-world business intelligence reporting workflows
