## Project Overview  
This project analyzes the 2024 data job market with a focus on Data Analyst roles. The dashboard is designed to help users understand hiring trends, salary distribution, and role-specific employment patterns using interactive visual analytics.

The dashboard enables dynamic exploration of job market data to support better career and market decision-making.

---

## Project Objective  

The main goals of this project are to:

- Analyze hiring trends across data-related roles  
- Compare salary distribution between roles and job types  
- Identify employment patterns such as remote work and degree requirements  
- Visualize geographic distribution of job opportunities  
- Enable detailed role-level exploration using drill-through functionality  

---

## Key Features  

- Interactive job role selection using slicers  
- KPI cards for job volume and salary overview  
- Monthly hiring trend analysis  
- Salary comparison across selected roles  
- Geographic visualization of job postings  
- Drill-through navigation for role-specific insights  

---

## Tools & Skills Used  

- Power BI Desktop  
- Interactive Dashboard Design

## How to Use  

1. Select a Job Role using the slicer  
2. View updated KPIs for salary and job count  
3. Analyze monthly hiring trends  
4. Explore salary distributions and geographic patterns  
5. Use drill-through to access role-level details  

---

## Key Insights  

- Data Analyst job postings declined after August and recovered toward December 2024  
- Median yearly salary for Data Analyst roles is approximately $90K  
- Hourly and yearly salaries show strong correlation patterns  
- Full-time positions dominate the job market  
- Remote work availability varies across job postings  
- Job demand is concentrated in major technology regions
