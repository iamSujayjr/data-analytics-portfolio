# Data Job Salary Analysis Dashboard (Excel)
![excel (1)](https://github.com/user-attachments/assets/9514880d-447d-4351-a3a2-f624642fc135)

## Overview
This project is an interactive Excel dashboard designed to help job seekers explore salary trends, job availability, and hiring platforms across different data roles, countries, and job types. The dashboard allows users to dynamically filter job market data and instantly view updated insights.

## Project Objective
The main goal of this project is to:
- Analyze salary trends for data-related roles
- Compare job demand across roles and job types
- Identify top job platforms for targeted job searching
- Provide an easy-to-use interactive salary exploration tool

## Key Features
- Dropdown-based filters for:
  - Job Title  
  - Country  
  - Job Type (Full-time, Contract, etc.)
- Automated KPI calculations:
  - Median Salary
  - Total Job Count
  - Top Hiring Platform
- Interactive charts for salary comparison
- Geographic visualization for country-based insights
- Data validation to prevent incorrect user inputs
- Protected backend sheets to maintain data integrity

## How to Use
1. Select a Job Title using the dropdown menu  
2. Choose a Country for location-specific insights  
3. Select Job Type (Full-time, Contract, etc.)  
4. Review updated salary metrics, job count, and hiring platforms  
5. Analyze charts to compare salary trends across roles  

## Key Insights
- Salary levels vary significantly based on job role and location  
- Full-time roles generally show higher salary stability  
- Certain platforms consistently dominate job postings  
- Interactive filtering allows users to tailor insights based on personal career goals  

## Notes
- Only input cells and dropdowns are editable  
- Backend calculation sheets are protected to avoid accidental data modification  
- Designed to simulate real-world business dashboard usage
