# 📊 Data Job Market Analysis (Python)

## 📌 Project Overview
This project analyzes a large dataset of job postings (700k+ rows) to understand trends in the data job market.

The analysis focuses on job demand, salary patterns, and role distribution using Python.

---

## 🎯 Objectives
- Identify the most common data roles
- Analyze salary distribution across roles
- Understand demand by country
- Explore the effect of experience level on salary

---

## 🛠 Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
